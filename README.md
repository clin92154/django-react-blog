# Django&React Blog

### Description
Django Rest Fremework建立API、React架設部落格，為練習用途。
- 實現功能如下:
  - 使用JWT及Django加密middleware建立會員登入機制
  - 會員發文、留言、按讚功能
  - 會員個人資料更新

### Framework
- Backend
  - Django
  - Django Rest Framework
- Frontend
  - React.js
